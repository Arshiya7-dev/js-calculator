# 🧮 JS Calculator

A simple and beautiful calculator with a modern Glassmorphism design, built with pure **HTML**, **CSS**, and **JavaScript** — no frameworks or extra libraries.

## ✨ Features

- 🎨 Modern Glassmorphism design with blur and glow effects
- ➕ ➖ ✖️ ➗ Four basic math operations
- ✅ Input validation (highlights empty fields with an error)
- 🚫 Division by zero prevention
- 💫 Smooth entrance and hover animations
- 📱 Responsive design

## ▶️ How to Run

To run the project locally:

```bash
git clone https://github.com/arshiya7-dev/js-calculator.git
cd js-calculator
```

Then open the `index.html` file in your browser. No packages or dependencies needed.
